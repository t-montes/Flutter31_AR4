Description

Blackhole is an online music streaming that allows you to listen music from Spotify, Youtube and their own music catalog. This app is created to promote the high-resolution music and forget about expensive subscriptions. It is an open-source app and features a user-friendly UI, custom themes, free downloads, multiple language support and cool animations.

There are several features integrated in the application, as well as a variety of technologies that make it possible to achieve the desired functionalities. In the following sections, we will analyze the most important aspects of the application, such as the architecture, the design, the techniques used and the performance.

Blackhole sets itself apart in the music streaming industry by offering unique functionalities and a fresh approach to standard features. It provides users with the flexibility to modify song playback speed, free downloads, and seamless integration with popular platforms like Spotify and YouTube Music, all while sourcing music from a diverse array of channels.

Advanced Playback Controls

Blackhole offers advanced playback control features, allowing users to adjust the speed of songs to suit their listening preferences. This customization enhances the user experience, catering to varied musical tastes and scenarios.

Seamless Integration and Free Downloads

Users can effortlessly download their favorite tracks and integrate playlists from platforms like Spotify and YouTube Music, all at no cost. This feature bridges the gap between multiple streaming services, providing a comprehensive music collection in one place.

At the heart of Blackhole's innovation lies its open-source nature, inviting a community-driven approach to music streaming. This openness fosters continuous improvement and customization, enabling a platform that evolves with its users' needs.

Community-Driven Development

Leveraging the power of open-source, Blackhole benefits from the collective expertise of developers worldwide. This collaboration leads to innovative features and robust performance, tailored by a community passionate about music and technology.

Customizable User Experience

Thanks to its open-source framework, Blackhole offers unparalleled customization options. Users and developers can personalize the app, from interface tweaks to adding new functionalities, making each experience uniquely tailored.

Libraries used

Audiotagger

This library is used by BlackHole App to tag and manage metadata for audio files. This allows its core functionality to work, which is reproducing music files through the app.

Flutter/material.dart

This library used the main Flutter library in order to have powerful widgets and tools for building user interfaces based on Google's Material Design. Using this library, the BlackHole App makes sure to follow the material design guidelines that Google has created.

Flutter_gen/gen_l10n/
app_localizations.dart

BlackHole uses this library to access automatically generated translations for all the available languajes from the source code.

Hive/hive.dart

The app uses Hive, a real-time database that also has local storage for Flutter. BlackHole uses Hive for efficiently storing data on the device, such as application settings or user data.

Audio_service/ audio_service.dart

BlackHole uses this library for integrating easily audio into the applications. This library is necessary to reproduce the music selected from Youtube or Spotify in the background.

Get_it/get_it.dart

This library has the service of wrapper and dependency injection solution for Flutter. BlackHole app uses this for easily manage dependencies in the application, which makes having a better structure of code overall.
